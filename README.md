# Token Monitor Bot

Telegram бот для моніторингу нових токенів на Uniswap.

## Можливості

- Відстежує нові токени на Uniswap
- Показує символ, назву, контракт, ліквідність
- Фільтрує токени з ліквідністю < $1,000
- Миттєві алерти в Telegram

## Налаштування

### 1. Створи Telegram канал

1. Відкрий Telegram
2. Створи новий канал (не групу!)
3. Налаштування каналу → Інформація → Інвайт-посилання
4. Зроби канал **публічним** або додай бота як адміна

### 2. Створи Telegram бота

1. Знайди @BotFather в Telegram
2. Напиши `/newbot`
3. Введи назву бота (напр. `TokenAlertBot`)
4. Введи username (напр. `my_token_alert_bot`)
5. Збережи **токен бота** (для TELEGRAM_TOKEN)

### 3. Дай боту права адміна каналу

1. Відкрий канал → Налаштування → Адміністратори
2. Додай свого бота як адміністратора
3. Дай право "Публікувати повідомлення"

### 4. Отримай ID каналу

1. Додай бота @userinfobot
2. Напиши `/start`
3. Надішли повідомлення в канал
4. @userinfobot покаже ID каналу (для CHANNEL_ID)

### 5. Зареєструйся на Bitquery

1. Зайди на [bitquery.io](https://bitquery.io)
2. Зареєструйся (безкоштовно)
3. Отримай API ключ (для BITQUERY_API_KEY)

### 6. Встанови Python

1. Зайди на [python.org/downloads](https://www.python.org/downloads/)
2. Встанови Python 3.10+
3. Перевір: `python --version`

### 7. Запусти бота

```bash
# Перейди в папку бота
cd token-monitor-bot

# Встанови залежності
pip install -r requirements.txt

# Задай змінні середовища (заміни на свої)
export TELEGRAM_TOKEN="твій_токен_бота"
export CHANNEL_ID="твій_id_каналу"
export BITQUERY_API_KEY="твій_api_ключ"

# Запусти бота
python bot.py
```
